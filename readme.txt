== Installation ==
= This plugin can be installed directly from your site. =

1. Log in and navigate to Plugins & Add New.
2. Type \"OpenAI GPT3 Auto Product Description Writer\" into the Search input and click the \"Search\" button.
3. Locate the \"OpenAI GPT3 Auto Product Description Writer\" in the list of search results and click \"Install Now\".
4. Click the \"Activate Plugin\" link at the bottom of the install screen.


= It can also be installed manually. =
i) To install plugin directly from the admin area. Download the plugin to your computer, then log in to your WP admin area and go to Plugins > Add New.
ii) In the new page, click on “Upload Plugin” button on top and then click on “Choose File”. Locate the plugin zip file in your computer and then press “Install Now” button.
iii) Once everything is ready, in the next page, click on “Activate Plugin” button.
